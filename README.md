# FACIA - Face Liveness Detection

This is a sample project for testing [facia.ai](https://facia.ai/) service. For using this app you need a facia account(username and password).

## Instalation

`npm install`

Edit .env file with your own data

## Run

`npm run dev`


### Live Demo
Feel free to test out the [live demo](https://facia.vercel.app/) before dive into code.
